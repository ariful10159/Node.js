# Node.js File System (fs) Module

---

## ১. ফাইল সিস্টেম মডিউল import

```js
const fs = require('fs');
```
**ব্যাখ্যা:**
- এই লাইনে `fs` নামের একটি ভ্যারিয়েবলে Node.js এর বিল্ট-ইন ফাইল সিস্টেম (File System) মডিউলটি import করা হয়েছে।
- এটি ব্যবহার করে ফাইল তৈরি, পড়া, লেখা, আপডেট, ডিলিট ইত্যাদি ফাইল সংক্রান্ত কাজ করা যায়।
- সংক্ষেপে: Node.js এর ফাইল সিস্টেম মডিউলটি ব্যবহার করার জন্য কোডে যুক্ত করা হয়েছে।

---

## ২. ফাইল তৈরি ও লেখা (Asynchronous)

```js
fs.writeFile("demo1.txt", "My name is Ariful Islam", (err)=>{
    if(err){
        console.log(err);
    }else{
        console.log("Successful")
    }
});
```
**ব্যাখ্যা:**
- এই কোডটি দিয়ে `demo1.txt` নামে একটি ফাইল তৈরি করা হচ্ছে এবং তাতে "My name is Ariful Islam" লেখা হচ্ছে।
- এটি asynchronous (অ্যাসিনক্রোনাস) পদ্ধতিতে কাজ করে—মানে, কাজটি শেষ হলে নিচের ফাংশনটি (callback) চালু হয়।
- যদি কোনো সমস্যা (error) হয়, তাহলে error মেসেজ দেখাবে。
- যদি সফল হয়, তাহলে "Successful" মেসেজটি দেখাবে。
- সংক্ষেপে: নতুন ফাইল তৈরি ও লেখা যায়, এবং কাজ সফল হলে বা সমস্যা হলে তা জানানো হয়।

## ৩. ফাইলের নাম পরিবর্তন

```js
fs.rename("demo1.txt", 'demo2.txt', (err)=>{
    if(err){
        console.log(err);
    }else{
        console.log("successfully renamed")
    }
})
```
**ব্যাখ্যা:**
- এই কোডটি দিয়ে `demo1.txt` ফাইলের নাম পরিবর্তন করে `demo2.txt` রাখা হচ্ছে。
- এটি asynchronous ভাবে কাজ করে—নাম পরিবর্তন শেষ হলে callback চালু হয়。
- যদি কোনো সমস্যা হয়, তাহলে error মেসেজ দেখাবে。
- সফল হলে "successfully renamed" মেসেজ দেখাবে。
- সংক্ষেপে: ফাইলের নাম পরিবর্তন করা যায় এবং কাজ সফল হলে বা সমস্যা হলে তা জানানো হয়।

## ৪. ফাইল ডিলিট (মুছে ফেলা)

```js
fs.unlink('demo2.txt', (err)=>{
    if(err){
        console.log(err);
    }else{
        console.log("successfully deleted")
    }
})
```
**ব্যাখ্যা:**
- এই কোডটি দিয়ে `demo2.txt` ফাইলটি ডিলিট (মুছে ফেলা) করা হচ্ছে。
- এটি asynchronous ভাবে কাজ করে—ফাইল ডিলিট শেষ হলে callback চালু হয়。
- যদি কোনো সমস্যা হয়, তাহলে error মেসেজ দেখাবে。
- সফল হলে "successfully deleted" মেসেজ দেখাবে。
- সংক্ষেপে: ফাইল ডিলিট করা যায় এবং কাজ সফল হলে বা সমস্যা হলে তা জানানো হয়।

## ৫. ফাইলের অস্তিত্ব যাচাই

```js
fs.exists('demo2.txt', (result)=>{
    if(result){
        console.log("found");
    }else{
        console.log("not found")
    }
})
```
**ব্যাখ্যা:**
- এই কোডটি দিয়ে চেক করা হচ্ছে `demo2.txt` ফাইলটি আছে কিনা。
- এটি asynchronous ভাবে কাজ করে—চেক করা শেষ হলে callback চালু হয়。
- যদি ফাইলটি থাকে, তাহলে "found" মেসেজ দেখাবে。
- না থাকলে "not found" মেসেজ দেখাবে。
- সংক্ষেপে: কোনো ফাইলের অস্তিত্ব আছে কিনা তা যাচাই করা যায় এবং ফলাফল অনুযায়ী মেসেজ দেখানো হয়।
